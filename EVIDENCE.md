# Evidence — Contract Quest, built with watsonx only

Verifiable record of the build: every screen of the game was generated by an LLM on
IBM watsonx.ai, through GitPilot, under Matrix Builder contracts.

| | |
|---|---|
| Contract engine | `agent-generator` 0.2.0 (`mb`) |
| Coder driver | GitPilot — `gitpilot generate`, `GITPILOT_PROVIDER=watsonx` |
| Provider | **IBM watsonx.ai** (`us-south.ml.cloud.ibm.com`, `/ml/v1/text/chat`) |
| Model | **`openai/gpt-oss-120b`** |

Each batch: `mb next` → `mb prompt --coder gitpilot` → `gitpilot generate` (gpt-oss-120b,
given the contract + the current file + the batch spec) → `mb check`. Allow-list every batch:
**`frontend/index.html` only**.
